# CSV to postgreSQL Migration Project

## Project Overview 
A Python-based data engineering project that automates the ingestion, validation, and migration of CSV data into a PostgreSQL database using secure environment configuration and error handling.

### What the Project Does
* Cleans and validates the data
* Creates or maps PostgresSQL tables
* Insert records efficiently into database
* Handles errors such as invalid data
* Keeps credentials secure using .env file

### Tools Used
* Python
* PostgreSQL
* Psycopg2
* pandas 
* dotenv
* VS Code

### Key Features 
* Secure database connection using environment variables 
* Error handling with try/except blocks 
* version-controlled workflow
